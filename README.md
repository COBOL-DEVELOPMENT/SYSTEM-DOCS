# SYSTEM-DOCS (COBOL / Mainframe System Analysis)

## Overview
This project focuses on analyzing and documenting COBOL-based legacy systems used in enterprise environments such as banking and insurance.  
It demonstrates the ability to understand complex systems, interpret business logic, and produce clear technical documentation.

---

## Project Goals
- Break down legacy COBOL systems into understandable components  
- Document data flow, program structure, and job execution  
- Improve maintainability and support future system modernization  

---

## Technical Skills
- COBOL (Mainframe Application Development)  
- JCL (Batch Job Processing)  
- DB2 (Relational Database)  
- System Analysis & Reverse Engineering  
- Technical Documentation
